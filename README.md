## Private / NDA Work

Some project code, datasets, model weights, and partner-specific details are private due to research collaboration agreements.

**MissenseImpact — Population-Aware Genomic Variant Effect Predictor**
- Built a scalable pipeline to process raw VCF files into a curated, model-ready dataset
- Fine-tuned ESM-2 150M using masked-variant scoring inspired by AlphaMissense
- Evaluated population-aware fine-tuning for missense variant classification in underrepresented South Asian ancestries
- Outperforms AlphaMissense on the South Asian evaluation set based on AUPRC and AUROC
- Code, data, model weights, and partner-specific details are protected under MOU/NDA
